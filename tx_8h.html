◆ start_stop()

void start_stop ( String  Op)

Función que parte del STOP. Permite detener la señal y cambiar los valores de la frecuencia y dureza.

Parámetros
OpAcción a ejecutar
Devuelve
NONE No retorna nada.

Función que trabaja con variables globales que cambian el estado del sistema cuando se está detenido.

Si el sistema está detenido y se repite la misma Acción, de DUTY o FREQ, este no cambia de estado.

Si el sistema está detenido y se meten dos Acciones seguidas dierentes, DUTY y luego FREQ, cambia el estado del sistema.
